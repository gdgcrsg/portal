## 📝 About
This portal is designed to manage and showcase the activities of the Gurudayal Govt. College Rover Scout Group. It provides easy access to important information, member directories, and group resources.

## ✨ Features
- **Member Directory**: Browse and search for group members
- **Patrol Team Management**: View different patrol teams and their members
- **Admin Section**: Administrative interface for managing content
- **Admission Information**: Details about joining the group
- **Blood Donors Registry**: Special section for blood donor members
- **BSID Verification**: Scout ID verification system
- **48th Anniversary**: Special celebration page

## 📄 Pages
- **Home** (`index.html`): Main landing page with portal navigation
- **Members** (`members.html`): Complete member directory with patrol team information
- **Admin** (`admin.html`): Administrative panel with archived member management
- **Admission** (`admission.html`): Information for prospective members
- **Blood Donors** (`blood-donors.html`): Directory of blood donor members with avatar styling
- **BSID Verify** (`bsid-verify.html`): Scout ID verification system
- **48th Anniversary** (`48th-anniversary.html`): Special event page

## 🛠️ Technologies Used
- HTML5
- CSS3 (with hover effects and modern styling)
- JavaScript
- GitHub Pages (hosting)
