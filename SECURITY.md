## 🐛 Reporting a Vulnerability

We take the security of our Rover Scout group portal seriously. If you discover a security vulnerability, please follow these steps:

### **DO NOT** create a public GitHub issue for security vulnerabilities

### Instead, report it privately:

1. **Contact the Development Team:**
   - Email: rovermdashfaq@gmail.com
   - Or reach out directly to any website maintainer

2. **Include in your report:**
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (if available)

3. **What happens next:**
   - We'll acknowledge receipt within 48 hours
   - We'll investigate and provide a timeline for fixes
   - We'll keep you updated on progress
   - We'll credit you (if desired) when the issue is resolved

## 🔐 What to Report

Please report:
- XSS (Cross-Site Scripting) vulnerabilities
- SQL injection risks
- Authentication/authorization issues
- Data leaks or exposure
- Broken access controls
- Any other security concerns

## ✅ What We Promise

- We'll respond promptly to all valid reports
- We'll fix verified vulnerabilities quickly
- We'll never take legal action against ethical researchers
- We'll publicly acknowledge your responsible disclosure (with permission)
